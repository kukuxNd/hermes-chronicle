# Hermes Chronicle

## 一句话
「你永远不知道上次你改变了什么——直到你回来。」

## 核心创新
**持久记忆引擎**：Hermes Chronicle 不是普通的文字冒险。它用 Hermes Agent 的记忆系统驱动 NPC 的「人生」。NPC 记得你上次来是什么时候、说了什么、做了什么。他们会提起：「你还记得吗，上次你……」

## 游戏世界设定

### 主题
蒸汽朋克 × 记忆都市 — 一座因为 AI 而拥有记忆的城市

### 主线剧情
城市中有一座「记忆塔」，传说中存放着所有市民的记忆碎片。你的任务是进入记忆塔，揭开城市兴衰的真相。

### 核心 NPC（都有持久记忆）

**艾达（Ada）** — 记忆塔守门人
- 她记得每一个进入过记忆塔的人，包括他们问了什么问题、得到了什么答案
- 如果你第二次进入，她会说：「你回来了。上次你问的那件事，后来我想了很久。」

**维克多（Victor）** — 蒸汽工程师
- 他记得你给他提过的技术建议
- 如果你教了他一个更好的方案，下次见面他会用上了

**沉默者（The Silent）** — 记忆塔最深处的存在
- 只有你的最终选择能与他对话
- 他记得整座城市所有人的记忆

## 技术架构

### 数据层
- SQLite 数据库存储游戏状态
- 每个玩家有独立的 session
- 每个 NPC 有独立的「记忆表」，记录与玩家的每次交互

### AI 层
- Hermes Agent 驱动 NPC 的「灵魂」
- 每个 NPC 有独特的 system prompt，决定他们的性格和记忆表达方式
- 游戏主持人（Game Master）用 Hermes 的 Skills 系统管理剧情分支

### 交互层
- 终端界面（主）：浸入式文字冒险
- Web 界面（辅）：状态查看 + 存档管理
- 每个 NPC 对话都是一次真实的 LLM 调用

## 差异化为什么强

普通文字游戏 = if-else 剧本
Hermes Chronicle = AI 真正理解你的选择，并记住一切

这和 Hermes Agent 的核心理念完全一致：「The agent that grows with you」
